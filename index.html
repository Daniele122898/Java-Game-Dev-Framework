Java-game-dev-framework

This is a very simple but effective Java Game Dev Framework

View on GitHub Download .zip Download .tar.gz

Commented on every major line

I tried to comment on every major line of code so that it would be as easy as possible to understand what each line does. The Code is also written clean and I used more Classes then needed to make it easier to follow and understand.

Aim

The aim of this Framework is to:

  • Give newcomers an easy Framework to work with and instantly start programming their games
  • Lines are commented for the newcomers to understand each step and thus create their own framework or expand this one.

Features

As a Game Development Framework it features everything you need to create easy games. There is an input handler, game loop (which you can edit the FPS you want by changing the thread.sleep(x) time, game thread, Jframe with changable name and res ofcourse, States and Resource management. Everything you need ;)
